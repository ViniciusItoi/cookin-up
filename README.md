# cookin-up

Aplicacao web para busca de receitas com base nos ingredientes selecionados.

Projeto do curso de Vue.js que fiz para praticar componentes, props, eventos e consumo de dados em uma aplicacao com Vue 3 + TypeScript.

## Tecnologias

- Vue 3
- TypeScript
- Vite

## Funcionalidades

- Selecao de ingredientes
- Listagem de receitas compativeis
- Mensagem de estado vazio quando nao ha resultados

## Como executar

```sh
npm install
npm run dev
```

## Build de producao

```sh
npm run build
```
